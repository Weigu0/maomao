# PurrHaus

PurrHaus 是一个面向国际市场的猫用品静态电商首页示例，展示猫家具、护理用品、玩具、喂食设备等商品，并包含基础购物车交互。

## 功能

- 响应式首页布局，适配桌面端和移动端。
- 商品集合、精选商品、物流信任信息和护理文章展示。
- 商品分类筛选：全部、家具、护理、玩具。
- 购物车抽屉：添加商品、增减数量、移除商品、计算小计。
- Newsletter 表单提交反馈。
- 使用 Unsplash 远程图片资源，无需本地图片依赖。

## 本地运行

这是一个纯 HTML/CSS/JavaScript 项目，不需要安装依赖或执行构建。

直接用浏览器打开：

```bash
open index.html
```

也可以用任意静态文件服务器运行，例如：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 项目结构

```text
.
├── index.html   # 页面结构与内容
├── styles.css   # 页面样式、响应式布局和购物车样式
└── script.js    # 商品筛选、购物车和提示消息交互
```

## 技术栈

- HTML5
- CSS3
- 原生 JavaScript

## 说明

当前项目是前端静态原型，购物车数据仅保存在页面内存中，刷新后会重置。Checkout 按钮未接入真实支付流程。
